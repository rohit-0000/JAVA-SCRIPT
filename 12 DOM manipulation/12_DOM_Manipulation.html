Window

Represents the browser window or tab in which the JavaScript code is running.

DOM

  • Document Object Model (DOM):
  • represented as an object.
  • The DOM allows JavaScript code to interact with and manipulate elements of the HTML page.
  • Access to the DOM is obtained through the document object.

Document Object:

  • The root object of the DOM, representing the entire HTML document.
  • Provides methods and properties to access and manipulate elements, attributes, and text nodes within the document.
  • EX:
    • document.getElementById("Fpara");
    • document.getElementsByClassName("Para");
    • document.getElementsByTagName("p");
    • document.querySelector("#Fpara");
    • document.querySelector(".Para"); // only topmost para with class "Para" is selected.
    • document.querySelectorAll(".Para"); // all with class "Para" is selected.
    • document.querySelectorAll("h1");

Browser Object Model (BOM):

  • A set of objects that provide access to the browser's environment and features, beyond the content of the web page.
  • Includes objects like window, location, history, screen, navigator, and more.
  • Allows JavaScript code to interact with the browser's features, such as opening new windows, controlling history, and accessing device information.

Key Points:

  • The Window object represents the browser window.
  • The DOM represents the structure of the HTML document.
  • The document object is the root of the DOM.
  • The BOM provides access to the browser's environment and features.
  • JavaScript code can use these objects to interact with the browser and manipulate the content of web pages.
